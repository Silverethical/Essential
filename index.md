# Description
اگر از دستگاه جبلبریک شده استفاده می کنید، صددرصد روزی می رسد که مجبور شوید دستوری در ترمینال وارد کنید. اِسِنشال این پروسه را برای شما آسان تر می کند. از دستورات ساده ای مانند ریسپرینگ و سیف مد گرفته تا دستورات پیچیده تری مانند رفع ارورهای سیدیا همه و همه با فشردن چند دکمه قابل اجرا می باشند  


<br/>
<br/>


# What Essential can do
* Respring  
*ریسپرینگ* 

* Safe Mode  
*حالت امن* 

* Other commands  
*سایر دستورها*  

   * Uptime  
   *مدت زمان روشن بودن دستگاه*  
   
   * UICache  
   *اگر پس از نصب یا حذف یک برنامه یا توییک آیکون آن اضافه یا پاک نشد، این گزینه را انتخاب کنید*  
   
   * LDRestart  
   *نوعی ریبوت بدون از دست دادن جیلبریک*  
   
   * Reboot userspace  
   *همان گزینه بالا ولی جدیدتر و احتمالا بهتر*  
   
   * Kill homed  
   *رفع مشکلاتی مانند ناپدید شدن هوم مجازی، از کار افتادن اینترنت سیم‌کارت و غیره*  
   
   * Reboot  
   *ریبوت دستگاه*  
   
   * Power off  
   *خاموش کردن دستگاه*  
   
   * Set Generator (Checkra1n)  
   *تنظیم نانس روی چکرین*  
   
   * Run command(s) as root  
   *اجرای دستور دلخواه*  

* Tweak Management  
*مدیریت توییک‌ها*  

   * Upgrade Tweaks  
   *آپدیت توییک‌ها*  
   
   * DEB installer  
   *نصب فایل دب از طریق لینک*  
   
   * Export tweak list  
   *استخراج لیست توییک‌های نصب شده*  

* Source Management  
*مدیریت سورس‌ها*  

   * Refresh Sources  
   *رفرش سورس‌ها*  
   
   * Export sources  
   *استخراج سورس‌ها*  
   
   * Import sources  
   *وارد کردن سورس‌ها*  
   
   * Revert sources  
   *Import برگرداندن سورس‌ها به قبل از*  

* Packages Managers  
*پکیج‌منیجر ها*  

   * Fix cydo error  
   *در سیدیا cydo رفع ارور*  
   
   * Fix Cydia crash  
   *رفع کرش سیدیا برای برخی کاربران*  
      *metadata.cb0  با حذف فایل*  

   * Install Cydia/Zebra/Installer 5  
   *نصب سیدیا / زبرا / اینستالر 5*  
   
   * Remove Zebra/Installer 5  
   *حذف زبرا / اینستالر 5*  

* About  
*درباره*  

   * Check for updates  
   *چک کردن برای نسخه جدید*  

   * Report bugs  
   *گزارش باگ*

   *حمایت مالی از سازنده*
